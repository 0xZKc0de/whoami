Built a specialized platform for engineering ATS-optimized resumes, utilizing AI to maximize candidate compatibility with automated screening systems.

### Key Features
- **AI Content Generation**: Uses GPT-4 to suggest role-specific bullet points.
- **Real-time Preview**: See changes instantly as you type.
- **PDF Export**: High-quality PDF generation with selectable text.
- **Templates**: Multiple professional templates to choose from.

The project was built to solve the pain point of repetitive resume tailoring for job applications. It currently serves over 500 active users.
