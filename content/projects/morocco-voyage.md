A complete travel management and reservation platform designed specifically for the Moroccan tourism market.

### Technical Overview
- **Enterprise Backend**: Built with Spring Boot, ensuring a robust and scalable service layer with MySQL for reliable data persistence.
- **Dynamic Frontend**: Developed using Angular to provide a seamless, responsive experience across both Client and Admin interfaces.
- **Secure Payments**: Integrated with the PayPal REST API to handle international and local transactions securely.
- **Quality Assurance**: Extensively tested with JUnit to ensure business logic reliability and system stability.

The platform streamlines the booking process, from destination discovery to secure checkout and reservation management.
